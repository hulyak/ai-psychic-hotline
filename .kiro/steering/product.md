# Product Overview

AI Psychic Hotline is a spooky fortune-telling web application that combines tarot card readings with AI-generated fortunes and movie recommendations.

## Core Features

- **Three Mystical Realms**: Users choose between Love, Fate, or Shadows realms, each with distinct tone and personality
- **Tarot Card Readings**: Random draw of 3-5 cards with upright/reversed orientations
- **AI-Generated Fortunes**: Personalized readings based on user questions and drawn cards using Claude AI
- **Voice Input**: Speak your question using Whisper speech-to-text
- **Eerie Voice Output**: Hear fortunes spoken in spooky voices using OpenAI TTS
- **AI-Generated Card Art**: Optional DALL-E 3 generated unique tarot card images
- **Movie Oracle**: Contextual horror/thriller movie recommendations matched to the reading
- **Atmospheric UI**: Dark, mystical interface with 3D smoke effects and spooky aesthetics

## User Flow

1. Select a realm (Love, Fate, or Shadows)
2. Enter a question for the spirits
3. View drawn tarot cards with orientations
4. Receive AI-generated fortune and movie recommendation
5. Option to draw new reading or change realms

## Tone & Voice

The application maintains a mystical, eerie atmosphere throughout. Fortunes are cryptic and spooky, avoiding modern slang. Error messages stay in character ("The veil between worlds has torn").
